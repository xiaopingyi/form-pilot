# 贡献指南

感谢你对 FormPilot 的关注！

## 如何贡献

### 报告 Bug

1. 先搜索 [已有 Issues](../../issues) 确认问题未被报告
2. 使用 [Bug 报告模板](../../issues/new?template=bug_report.md) 提交问题
3. 提供尽可能详细的信息：
   - 复现步骤
   - 环境信息（浏览器版本、操作系统、UI 框架）
   - 控制台错误信息
   - 截图

### 功能建议

1. 先搜索 [已有 Issues](../../issues) 确认建议未被提出
2. 使用 [功能建议模板](../../issues/new?template=feature_request.md) 提交建议
3. 描述清楚使用场景和期望的解决方案

### Issue 规范

- 标题简洁明了，使用中文
- 一个 Issue 只描述一个问题或建议
- 提供足够的上下文信息
- 保持礼貌和尊重
